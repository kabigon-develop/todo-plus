## 2. 手工回归清单（核心链路）

### 2.1 任务（Todo）

1. 新增任务
- 操作：点击“新增任务”，填写标题/描述，保存。
- 预期：列表出现新任务。

2. 编辑任务
- 操作：点击“编辑”，修改标题，保存。
- 预期：列表展示更新后的标题。

3. 单条状态切换
- 操作：点击“完成/取消完成”。
- 预期：按钮文案和任务样式随状态变化。

4. 单条删除
- 操作：点击“删除”。
- 预期：任务从列表移除。

5. 批量完成
- 操作：勾选多条任务，点击“批量完成”。
- 预期：选中任务全部变为已完成。

6. 批量删除
- 操作：勾选多条任务，点击“批量删除”。
- 预期：选中任务全部被删除，批量按钮恢复禁用。

7. 搜索
- 操作：在“搜索任务标题或描述”输入关键字。
- 预期：列表按标题/描述关键字过滤。

8. 筛选
- 操作：切换“全部/进行中/已完成”。
- 预期：列表结果与筛选条件一致。

### 2.2 想法看板（Idea）

1. 新增想法
- 操作：切到“想法看板”，点击“新增想法”并保存。
- 预期：新想法出现在“想法”列。

2. 编辑想法
- 操作：点击想法卡片“编辑”，修改后保存。
- 预期：卡片内容更新。

3. 流程推进/回退
- 操作：点击“下一步/上一步”。
- 预期：卡片在“想法/评估/执行”三列间移动。

4. 转为任务
- 操作：把想法推进到“执行”后点击“转为任务”。
- 预期：按钮变禁用并显示“已转任务”；任务页出现对应任务。
